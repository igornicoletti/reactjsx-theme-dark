# Theme Dark

Vite + React + Tailwindcss

* [Getting Started | Vite](https://vitejs.dev/guide/)
* [Install Tailwind CSS with Create React App](https://tailwindcss.com/docs/guides/create-react-app)

## Installation

```
yarn install
```

## Usage
Development build

```bash
yarn dev
```

Production build

```bash
yarn build
```
